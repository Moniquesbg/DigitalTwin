# Aareon
Voor het bedrijf Aareon in Emmen heb ik samen met mijn klasgenoten de Digital Twin project opgepakt.

Voor dit project is er een webapplicatie gebouwd die alle ruimtes in het Aareon gebouw toont. De ruimtes zijn voorzien van sensoren die CO2, temperatuur, vochtigheid en bezetting kunnen meten. 
Met behulp van Zigbee wordt de data opgehaald.
